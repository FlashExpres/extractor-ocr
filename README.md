# Extractor de Datos Logísticos - ERS

Este sistema permite la extracción automatizada de datos desde imágenes (hojas de ruta, etiquetas) y texto plano utilizando IA (Gemini).

## Características
- 📄 **OCR Inteligente**: Extracción de datos desde fotos de hojas de ruta.
- 📋 **Procesamiento de Texto**: Limpieza y estructuración de datos pegados directamente.
- 💾 **Gestión de Clientes/Drivers**: Almacenamiento local de datos frecuentes.
- 📊 **Exportación**: Generación de archivos Excel (.xlsx) listos para logística.
- 🌓 **Modo Oscuro/Claro**: Interfaz adaptable y moderna.

## Cómo usar
1. Configura tu **Gemini API Key** en el panel de Ajustes.
2. Selecciona el Driver y el Cliente correspondiente.
3. Sube las imágenes o pega el texto en el área indicada.
4. Presiona "Cargar Datos" para procesar.
5. Edita los datos en la tabla si es necesario.
6. Exporta a Excel.

## Tecnologías
- HTML5 / CSS3 / JavaScript (Vanilla)
- API de Google Gemini (2.5-flash)
- SheetJS para exportación a Excel
- FontAwesome para iconografía
